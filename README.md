## 🚀 Deploy & Index Fast

### 1. Deploy
```bash
# Vercel
npx vercel --prod

# or Netlify CLI
netlify deploy --prod
```

### 2. Submit Sitemap to Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property
3. Go to **Sitemaps** → paste `https://yourdomain.vercel.app/sitemap.xml`

### 3. Ping Google Sitemap
Open in browser:
```
https://www.google.com/ping?sitemap=https://yourdomain.vercel.app/sitemap.xml
```

### 4. Request Indexing
In Google Search Console → **URL Inspection** → enter your URL → **Request Indexing**

### 5. Create Crawl Signals
Post your URL on:
- Reddit (r/BollywoodMovies, r/india, r/movies)
- Twitter / X with hashtags `#Dhurandhar #DhurandharTheRevenge`
- Quora answers about the movie
- Medium article about the film

Googlebot crawls these platforms frequently and will follow your links.

## 📢 Ad Monetisation

To activate Google AdSense:
1. Replace `ad-zone` placeholder divs with real `<ins class="adsbygoogle">` tags
2. Uncomment the AdSense script in `<head>`:
```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX" crossorigin="anonymous"></script>
```
3. Ensure content is original and not misleading (this site qualifies ✓)
